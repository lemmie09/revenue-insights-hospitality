# Revenue Insights Dashboard – Hospitality Domain (Atliq Grands)

A portfolio analytics project using 90 days of hotel revenue, occupancy, and booking-channel data to identify drivers behind declining performance and build an interactive Power BI dashboard with actionable insights.

---

##  Project Objective

Analyze operational performance for Atliq Grands to:

- Identify causes of revenue decline
- Evaluate pricing and occupancy performance
- Assess booking channel efficiency
- Provide data-backed recovery recommendations
- Simulate potential revenue uplift scenarios (10–20% improvement target)

---

## Key Business KPIs

The dashboard will track and analyze:

- **Total Revenue**
- **Occupancy %**
- **ADR (Average Daily Rate)**
- **RevPAR (Revenue per Available Room)**
- **Realization %**
- Weekday vs Weekend trends
- Booking channel contribution
- Property-level performance

##  Project Workflow

1. Raw data stored in `data/raw`
2. Data cleaning & preparation → saved to `data/processed`
3. Power BI model built using star schema
4. DAX measures created for core KPIs
5. Dashboard visuals developed (Executive + Drilldown views)
6. Insights documented in `/docs`
7. Screenshots added to `/screenshots`
8. All updates tracked via GitHub
